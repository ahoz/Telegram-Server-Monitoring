# Telegram-Server-Monitoring
Send User Login and Nagios Plugin Output (Apt, Disk) to Telegram

**How to use it:**

 1. Clone to Server
 2. Set API Key and ChatID
 3. Change Disk Settings (checkdisks dict variable)
